# Twitter 互关检测助手 (Twitter Unfollow Helper)

中文 | [English](./README.md)

这是一个强大的 JavaScript 浏览器脚本（支持油猴/控制台），旨在帮助 Twitter (X) 用户快速、直观地在“正在关注”列表页面筛选出那些**没有回关你**的用户。

## 🌟 功能特点

- **全自动检测**：实时监测 Twitter 页面上的“关注了你”标签。
- **视觉增强与过滤**：
  - **未回关用户**：背景标红并加上醒目的红边框，方便一眼识别。
  - **互关用户**：默认半透明弱化，让你专注于目标。
- **物理“清理”模式**：可开启特殊模式，彻底**从视图中**移除互关用户的格子，使列表里剩下的全是未回关用户。
- **智能适配滚动**：自动平滑向下滚动，持续探测大容量关注列表。
- **交互式控制面板**：页面右上角提供浮动面板，包含“开始/恢复”、“暂停”及“清理列表”功能。
- **持久标记**：采用 CSS 注入和属性追踪技术，有效对抗 Twitter 的 DOM 回收机制，确保标记不闪烁、不丢失。
- **多语言支持**：完美支持中文（简体/繁体）和英文界面。
- **安全性**：纯视觉辅助工具，**不执行**任何自动化 API 调用或批量取关动作，零封号风险。

## 🚀 安装步骤

### 方式一：作为用户脚本安装 (推荐)
1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 等脚本管理器。
2. [点击此处从 Greasy Fork 安装]() *(正在发布中...)*。

### 方式二：浏览器控制台
1. 复制本项目中的 `twitter_unfollow_helper.js` 全部代码。
2. 打开 Twitter 正在关注页面：`https://twitter.com/[你的用户名]/following`。
3. 按 `F12` 打开开发者工具，在 **Console (控制台)** 中粘贴代码并回车运行。

## 🛠 使用说明

1. 进入 Twitter/X 的 **正在关注 (Following)** 页面。
2. 查看页面右上角的控制面板。
3. 点击 **开始/恢复** 按钮启动自动滚动和筛选。
4. 点击 **清理列表** 即可隐藏所有互关用户，只留下红色的待确认名单。
5. 如果需要取关某人，可以点击 **暂停**，手动取关后再点击 **恢复** 继续扫描。
